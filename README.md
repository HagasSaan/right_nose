# Right nose
Please don't ask why it is named like so

### Backend
1. Install [docker](https://docs.docker.com/engine/install/)
2. Install [rust](https://www.rust-lang.org/tools/install)

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
fastapi run main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
